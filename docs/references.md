## 분석

### 1. Craft Brewer
- **성격:** 홈브루잉 전문 쇼핑몰. 맥주 재료·장비 중심
- **가격대:** 1,500원(재료) ~ 100만원대(장비)
- **구조:** 커머스 + 아카데미·공방·커뮤니티 병행 — SOWN의 커머스+클래스 구조와 유사
- **디자인:** 모던, 상품 그리드 중심. 감성 약함
- **시사점:** "발효판 크래프트브루어" 포지션 가능. 단 감성·큐레이션이 없음 → SOWN의 차별화 지점

### 2. Beer School
- **성격:** 홈브루잉 재료·장비 + 클래스 특화
- **가격대:** 5,000원 ~ 220만원대
- **슬로건:** "즐거운 맥주만들기의 세상에 오신 것을 환영합니다"
- **디자인:** 흰 배경 그리드. 정보 전달 중심. 감성 없음. 오래된 느낌
- **시사점:** 클래스 연계 구조 참고. 브랜딩은 반면교사

### 3. _soolow (술로우)
- **성격:** 전통주 담그기 클래스 + 감성 라이프스타일 브랜드
- **채널:** 인스타그램 @_soolow + 네이버 블로그 + 카카오 예약
- **컬러:** 베이지(#fffcf3) + 갈색(#9a823b) — 따뜻하고 고급스러운 톤
- **구조:** 클래스 예약 + 굿즈(잔) 판매 병행. 스토리텔링 중심
- **시사점:** SOWN이 가장 가까이 봐야 할 레퍼런스. 감성·클래스 구조 유사. 단, 커머스(장류·도구) 규모는 SOWN이 훨씬 큼

### 4. 하마하마브루잉 / 5. 어떤하루
- 네이버 스마트스토어 — 자동 분석 불가 (네이버 차단)
- 직접 접속 후 분석 필요
