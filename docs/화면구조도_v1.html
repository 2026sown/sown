고객 화면 — Phase 1 필수
브랜드 · 탐색
01 Home 필수
  • 히어로 섹션 — 브랜드 슬로건 + 풀블리드 비주얼
  • 추천 상품 — 큐레이션 picks 4~6개
  • 카테고리 진입 — 장류 / 도구 / 술담그기
  • 브랜드 스토리 — 발효·기다림 컨셉 한 줄
  • Class 진입 CTA — 체험 클래스 소개
  • 신상품 / 시즌 배너
연결: Shop 목록 · Product Detail · Class
02 Shop 목록 필수
  • 카테고리 탭 — 전체 / 장류 / 발효 도구·재료 / 술 담그기 / 선물 세트
  • 정렬 — 신상 / 인기 / 가격순
  • 상품 그리드 — 이미지 + 이름 + 가격 + 큐레이션 한 줄
  • 품절 표시
  • 페이지네이션 or 무한스크롤
연결: Product Detail · 검색 결과
03 Product Detail 필수
  • 상품 이미지 — 멀티 이미지 / 줌
  • 상품명 + 가격
  • 큐레이션 노트 — SOWN 에디터 설명
  • 수량 선택 + 장바구니 / 바로구매
  • 상세 설명 — 원산지·용량·보관법
  • 연관 상품 추천
  • 배송 안내
연결: Cart · Checkout · Shop 목록
08 About 필수
  • 브랜드 스토리 — SOWN의 탄생 이유
  • 브랜드 철학 — 발효 = 기다림의 미학
  • 큐레이션 기준 — 어떻게 상품을 고르는가
  • 팀/창업자 소개
  • Shop CTA
연결: Home · Shop 목록
11 검색 결과 필수
  • 검색어 표시 + 결과 수
  • 상품 결과 목록 — Shop 목록과 동일 카드
  • 결과 없음 상태 — 빈 상태 + 추천 상품
  • 카테고리 필터
연결: Product Detail · Shop 목록
Class
04 Class (랜딩) 필수
  • 클래스 소개 — 장담그기 / 술담그기
  • 일정 안내 — 다음 기수 예정일
  • 커리큘럼 개요
  • 신청 CTA — Phase 1: 신청서 폼 or 카카오톡 연결
  • 후기
  • ⚠️ Phase 1 예약 시스템 없음 — Phase 2에서 개발
연결: Home · 외부 신청 채널
구매 플로우
05 Cart 필수
  • 담긴 상품 목록 — 이미지 + 이름 + 가격
  • 수량 변경 / 상품 삭제
  • 금액 요약 — 상품금액 / 배송비 / 합계
  • 무료배송 기준 안내
  • 결제하기 CTA
  • 빈 장바구니 상태 — 추천 상품
연결: Checkout · Product Detail
06 Checkout 필수
  • 주문 상품 요약
  • 배송지 입력 — 이름 / 연락처 / 주소 (우편번호 검색)
  • 배송 메모
  • 결제 수단 선택 — 카드 / 무통장 / 카카오페이 등
  • 최종 금액 확인
  • 결제하기 버튼 — PG 연동
연결: Order Complete · Cart
07 Order Complete 필수
  • 주문 완료 메시지
  • 주문 번호
  • 주문 요약 — 상품 / 금액 / 배송지
  • 마이페이지 이동 — 주문 내역 확인
  • 쇼핑 계속하기 CTA
연결: 마이페이지 · Home
계정 · 마이페이지
09 회원가입 / 로그인 필수
  • 이메일 로그인
  • 소셜 로그인 — 카카오 / 네이버 (검토)
  • 회원가입 — 이메일 / 비밀번호 / 이름 / 연락처
  • 비밀번호 찾기
  • 비회원 구매 — 체크아웃에서 선택 가능
연결: 마이페이지 · Checkout
10 마이페이지 필수
  • 주문 내역 — 날짜 / 상품 / 상태
  • 배송 조회 — 운송장 번호 연동
  • 취소 / 반품 신청
  • 개인정보 수정 — 비밀번호 / 연락처 / 주소
  • 배송지 관리 — 자주 쓰는 주소
  • 로그아웃
연결: Order Complete · Product Detail
어드민 화면 — 운영 필수
14 대시보드 운영 필수
  • 오늘 매출 / 주문 수 / 신규 회원
  • 처리 대기 주문 수 + 바로가기
  • 재고 부족 알림
  • 기간별 매출 차트 — 일/주/월
  • 최근 주문 목록
연결: 주문 관리 · 재고 관리 · 정산
15 상품 관리 운영 필수
  • 상품 목록 — 카테고리 필터 / 검색
  • 상품 등록 — 이름 / 가격 / 이미지 / 설명 / 큐레이션 노트
  • 상품 수정 / 삭제
  • 카테고리 관리
  • 노출 ON/OFF — 품절·숨김 처리
  • 대표 이미지 + 상세 이미지 업로드
연결: 재고 관리
16 주문 관리 운영 필수
  • 주문 목록 — 날짜 / 주문자 / 금액 / 상태
  • 상태별 필터 — 결제완료 / 준비중 / 배송중 / 완료 / 취소
  • 주문 상세 — 상품 / 배송지 / 결제 정보
  • 주문 상태 변경
  • 취소 / 환불 처리
  • 주문 검색 — 주문번호 / 이름 / 연락처
연결: 배송 관리 · CS/문의
17 배송 관리 운영 필수
  • 배송 전 주문 목록 — 운송장 미입력
  • 운송장 번호 입력 — 개별 / 엑셀 일괄 등록
  • 택배사 선택
  • 배송 현황 확인
  • 배송 완료 처리
연결: 주문 관리
18 회원 관리 운영 필수
  • 회원 목록 — 가입일 / 이름 / 이메일 / 주문 수
  • 회원 상세 — 주문 이력 / 개인정보
  • 회원 검색
  • 계정 정지 / 탈퇴 처리
연결: 주문 관리
19 재고 관리 운영 필수
  • 상품별 재고 수량
  • 재고 수동 입력 — 입고 / 조정
  • 입출고 이력
  • 재고 부족 알림 기준 설정
  • 품절 자동 처리
연결: 상품 관리 · 대시보드
20 정산 / 매출 운영 필수
  • 기간별 매출 합계 — 일 / 주 / 월 / 커스텀
  • 카테고리별 매출
  • 상품별 판매량 순위
  • 결제 수단별 통계
  • 엑셀 다운로드
연결: 대시보드
21 CS / 문의 운영 필수
  • 문의 목록 — 날짜 / 유형 / 처리 상태
  • 문의 유형 — 배송 / 교환반품 / 상품 / 기타
  • 답변 작성
  • 처리 완료 표시
  • 미처리 알림
연결: 주문 관리 · 회원 관리

  • 발효 이야기 · 레시피 · 큐레이션 아티클 목록
  • 카테고리 태그 필터
  • 그리드 레이아웃
13 Journal 상세 이후 추가

  • 연관 상품 연결 CTA
