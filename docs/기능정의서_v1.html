01 Home 필수
목적
  • 브랜드 첫인상 전달 + 핵심 상품·클래스 진입 유도
진입 경로
  • GNB 로고 클릭 / 도메인 직접 접근 / 외부 링크
컴포넌트
  • 히어로 섹션 — 풀블리드 비주얼 + 슬로건 + Shop CTA 버튼
  • 추천 상품 그리드 — 큐레이션 Picks 4~6개 (상품 카드)
  • 카테고리 진입 배너 — 장류 / 발효 도구·재료 / 술 담그기
  • 브랜드 스토리 한 줄 — 발효·기다림 컨셉
  • Class 진입 CTA — 체험 클래스 소개 + 신청 버튼
  • 신상품 / 시즌 배너 — 어드민에서 ON/OFF 가능
  • GNB — 로고 / Shop / Class / About / 검색 / 장바구니 / 마이페이지
  • Footer — 사업자 정보 / SNS / 이용약관 / 개인정보처리방침
기능
  • 추천 상품 클릭 → Product Detail 이동
  • 카테고리 배너 클릭 → Shop 목록 (카테고리 필터 적용)
  • Class CTA 클릭 → Class 랜딩 이동
  • 장바구니 아이콘 → Cart 이동 (담긴 수량 뱃지 표시)
상태
  • 로딩 — 상품 카드 skeleton 표시
  • 에러 — 상품 로드 실패 시 재시도 버튼
  • 빈 상태 — 추천 상품 없을 시 기본 카테고리 배너만 표시
나가는 경로
  • Shop 목록 / Product Detail / Class / About / Cart / 마이페이지 / 검색 결과
연결: Shop 목록 · Product Detail · Class · About
02 Shop 목록 필수
목적
  • 전체 상품 탐색 · 카테고리 분류 · 정렬로 구매 욕구 유도
진입 경로
  • GNB Shop 클릭 / Home 카테고리 배너 / 검색 결과 "전체 보기"
컴포넌트
  • 카테고리 탭 — 전체 / 장류 / 발효 도구·재료 / 술 담그기 / 선물 세트
  • 정렬 드롭다운 — 신상 / 인기 / 가격 낮은순 / 가격 높은순
  • 상품 카드 그리드 — 이미지 + 이름 + 가격 + 큐레이션 한 줄 + 품절 뱃지
  • 페이지네이션 or 무한스크롤 (구현 방식 결정 필요)
  • 결과 수 표시 — "총 N개 상품"
기능
  • 카테고리 탭 전환 → 해당 카테고리 상품 필터링
  • 정렬 변경 → 상품 목록 재정렬
  • 상품 카드 클릭 → Product Detail 이동
  • 품절 상품 — 카드 표시하되 클릭 비활성화
상태
  • 로딩 — 카드 skeleton 6개
  • 에러 — 로드 실패 메시지 + 재시도
  • 빈 상태 — "해당 카테고리 상품이 없습니다" + 전체 보기 버튼
나가는 경로
  • Product Detail / 검색 결과
연결: Product Detail · 검색 결과
03 Product Detail 필수
목적
  • 상품 정보 + 큐레이션 노트 제공으로 구매 전환
진입 경로
  • Shop 목록 카드 클릭 / Home 추천 상품 / 검색 결과 / 마이페이지 주문 이력
컴포넌트
  • 상품 이미지 갤러리 — 메인 + 썸네일 / 줌 기능
  • 상품명 + 가격
  • 큐레이션 노트 — SOWN 에디터 설명 (브랜드 차별화 핵심)
  • 수량 선택 스피너
  • 장바구니 담기 / 바로 구매 버튼
  • 상세 설명 탭 — 원산지 / 용량 / 보관법 / 제조사
  • 배송 안내 — 무료배송 기준 / 예상 도착일
  • 연관 상품 추천 — 같은 카테고리 4개
기능
  • 장바구니 담기 → Cart에 수량 추가 + 토스트 알림
  • 바로 구매 → Checkout 이동 (Cart 스킵)
  • 비로그인 장바구니 → 로컬스토리지 임시 저장 후 로그인 시 병합
  • 품절 상태 → 버튼 비활성화 + "품절" 텍스트
상태
  • 로딩 — skeleton UI
  • 에러 — 상품 없음 메시지 + Shop 목록 이동
  • 품절 — 버튼 비활성화 / 입고 알림 기능 (미정)
나가는 경로
  • Cart / Checkout (바로구매) / Shop 목록
연결: Cart · Checkout · Shop 목록
08 About 필수
목적
  • 브랜드 철학·스토리 전달로 신뢰 형성 및 충성 고객 전환
진입 경로
  • GNB About 클릭 / Home 브랜드 스토리 섹션
컴포넌트
  • 브랜드 스토리 — SOWN의 탄생 이유 + 대표 이미지
  • 브랜드 철학 — 발효 = 기다림의 미학 (핵심 카피)
  • 큐레이션 기준 — 어떻게 상품을 고르는가
  • 팀 / 창업자 소개
  • Shop 진입 CTA — "지금 쇼핑하기" 버튼
기능
  • CTA 클릭 → Shop 목록 이동
  • 인터랙션: 스크롤 리빌 애니메이션
상태
  • 정적 페이지 — 별도 로딩/에러 상태 없음
나가는 경로
  • Shop 목록 / Home
연결: Home · Shop 목록
11 검색 결과 필수
목적
  • 키워드 검색으로 원하는 상품 빠르게 탐색
진입 경로
  • GNB 검색 아이콘 → 검색어 입력 → 엔터 or 검색 버튼
컴포넌트
  • 검색어 표시 — '"OOO" 검색 결과 N개'
  • 상품 결과 목록 — Shop 목록과 동일한 카드 컴포넌트
  • 카테고리 필터 — 결과 내 재필터
  • 빈 상태 컴포넌트 — 추천 상품 4개
기능
  • 상품명 / 큐레이션 노트 텍스트 대상 검색
  • 검색어 변경 → 실시간 재검색 (debounce 300ms)
  • 카테고리 필터 클릭 → 결과 내 필터링
상태
  • 로딩 — skeleton 카드
  • 결과 없음 — "검색 결과가 없습니다" + 추천 상품
  • 에러 — 재시도 버튼
나가는 경로
  • Product Detail / Shop 목록
연결: Product Detail · Shop 목록

05 Cart 필수
목적
  • 담긴 상품 확인·수정 후 결제 진입
진입 경로
  • GNB 장바구니 아이콘 / 상품 상세 "장바구니 담기" 후 토스트
컴포넌트
  • 담긴 상품 목록 — 이미지 + 이름 + 가격 + 수량
  • 수량 변경 스피너
  • 상품 삭제 버튼
  • 금액 요약 — 상품금액 / 배송비 / 합계
  • 무료배송 기준 안내 — 진행 바 or 텍스트
  • 결제하기 CTA
  • 빈 장바구니 — 추천 상품 4개
기능
  • 수량 변경 → 금액 실시간 재계산
  • 상품 삭제 → 목록에서 제거 + 합계 재계산
  • 비로그인 → 로컬스토리지 Cart, 로그인 시 서버 동기화
  • 결제하기 클릭 → 비로그인이면 로그인 페이지로 리다이렉트
상태
  • 로딩 — skeleton 목록
  • 에러 — 재시도
  • 빈 상태 — "장바구니가 비어있습니다" + 추천 상품
나가는 경로
  • Checkout / Product Detail
연결: Checkout · Product Detail
06 Checkout 필수
목적
  • 배송지·결제 수단 입력 후 최종 결제 완료
진입 경로
  • Cart "결제하기" / Product Detail "바로 구매"
컴포넌트
  • 주문 상품 요약 — 이미지 + 이름 + 수량 + 가격
  • 배송지 입력 폼 — 이름 / 연락처 / 우편번호 검색 / 주소 / 상세주소
  • 배송 메모 — 드롭다운 + 직접 입력
  • 결제 수단 선택 — 신용카드 / 무통장입금 / 카카오페이 / 네이버페이
  • 최종 금액 확인 — 상품금액 + 배송비 + 합계
  • 결제하기 버튼 — PG 연동
기능
  • 우편번호 검색 → 카카오 주소 API 연동
  • 저장된 배송지 자동 불러오기 (로그인 시)
  • PG 결제 모듈 연동 (포트원 or 토스페이먼츠 — 결정 필요)
  • 결제 완료 → Order Complete 이동
  • 결제 실패 → 에러 메시지 + 재시도
상태
  • 로딩 — 결제 처리 중 스피너 + 버튼 비활성화
  • 에러 — 결제 실패 메시지 (코드별)
  • 빈 상태 — 상품 없으면 Cart로 리다이렉트
나가는 경로
  • Order Complete / Cart (취소 or 실패)
연결: Order Complete · Cart

  • 주문 완료 확인 + 마이페이지 이동 유도
진입 경로
  • Checkout 결제 성공 후 자동 이동
컴포넌트
  • 주문 완료 메시지 — 아이콘 + 텍스트
  • 주문 번호
  • 주문 요약 — 상품명 / 금액 / 배송지

  • 주문 번호 복사 버튼
상태
  • 정상 — 주문 완료 화면
  • 비정상 접근 — 주문 없으면 Home 리다이렉트
나가는 경로

연결: 마이페이지 · Home

09 회원가입 / 로그인 필수
목적
  • 회원 인증 후 구매·마이페이지 기능 사용 가능
진입 경로
  • GNB 마이페이지 / Checkout 비로그인 상태 / Cart "결제하기" 클릭
컴포넌트
  • 탭 전환 — 로그인 / 회원가입
  • 이메일 + 비밀번호 폼
  • 소셜 로그인 — 카카오 / 네이버 (도입 검토)
  • 비밀번호 찾기 링크
  • 회원가입 폼 — 이메일 / 비밀번호 / 이름 / 연락처 / 약관 동의
  • 비회원 구매 안내 — Checkout 연결 시 옵션 제공
기능
  • Supabase Auth 연동 — 이메일/소셜 로그인
  • 로그인 성공 → 이전 페이지 or 마이페이지 이동
  • 비밀번호 찾기 → 이메일 재설정 링크 발송
  • 회원가입 → 이메일 인증 후 활성화
상태
  • 로딩 — 버튼 스피너
  • 에러 — 잘못된 비밀번호 / 미가입 이메일 / 중복 이메일 메시지
나가는 경로
  • 마이페이지 / Checkout / 이전 페이지
연결: 마이페이지 · Checkout

상태

14 대시보드 운영 필수
목적
  • 운영 현황 한눈에 파악 + 긴급 처리 항목 빠른 접근
진입 경로
  • 어드민 로그인 후 기본 화면
컴포넌트

  • 재고 부족 알림 — 기준 이하 상품 목록

기능
  • KPI 실시간 갱신 (또는 새로고침)
  • 재고 부족 알림 클릭 → 재고 관리 이동
  • 최근 주문 클릭 → 주문 관리 상세
상태
  • 로딩 — 카드 skeleton
  • 데이터 없음 — "오늘 주문 없음" 등 메시지
나가는 경로
  • 주문 관리 / 재고 관리 / 정산·매출
연결: 주문 관리 · 재고 관리 · 정산
15 상품 관리 운영 필수
목적
  • 상품 등록·수정·삭제·노출 관리
진입 경로
  • 어드민 사이드바 "상품 관리"
컴포넌트
  • 상품 목록 테이블 — 이름 / 카테고리 / 가격 / 재고 / 노출 상태
  • 카테고리 필터 + 검색
  • 상품 등록 폼 — 이름 / 가격 / 이미지 / 설명 / 큐레이션 노트 / 카테고리
  • 이미지 업로드 — 대표 1장 + 상세 다수 (Supabase Storage)
  • 노출 토글 — ON / OFF / 품절
