## 공통 원칙

- 클리셰 블랙리스트 금지: "특별한 경험" / "프리미엄" / "최고의 품질" / "당신만을 위한" / "함께해요" / "지금 바로"
- 큐레이션 노트: SOWN 에디터가 이 상품을 고른 이유를 1~2문장으로 — 설명이 아니라 이유
- SD = 상품 상세 섹션 코드 (기능정의서 기준)
- 상품명 원칙: [종류/성격] + [구체 명칭/산지/특징] — 조사·수식어 최소화

## 카테고리 1 · 장류 (된장·간장·고추장)

### 상품명 작성 규칙
- **구조:** [종류] [원재료/산지/연수] (예: 재래식 된장 / 국산 콩 3년 숙성 간장 / 청양 고추장)
- **띄어쓰기 기준:** 종류 + 구체 명칭 사이만 한 칸
- **금지:** "정통", "전통의", "맛있는", "깊은 맛의" 등 형용사 남발
- **영문 병기 예시:** 재래 된장 · Doenjang

### 큐레이션 노트 (SOWN 에디터 한 줄)
- **원칙:** 이 상품을 고른 이유 — 재료·생산지·특징 중 가장 인상적인 한 가지
- **형식:** "~해서 골랐습니다." 또는 "~이기 때문입니다." 결말로 끝낼 것
- **예시 (된장):** "메주를 직접 띄우는 마지막 세대 장인의 손에서 나왔습니다. 3년 숙성, 냉장 보관이 필요 없습니다."
- **예시 (간장):** "국산 콩만 씁니다. 끓이지 않고 자연 발효한 간장이라 단맛이 없고 깊습니다."
- **예시 (고추장):** "찹쌀고추장입니다. 쌀의 단맛과 고추의 매운맛이 따로 놀지 않는, 균형 잡힌 맛을 위해 골랐습니다."

### SD-02 · Hook 카피 템플릿
```
[감각적 한 줄 — 맛/냄새/질감을 직접 묘사]

[이 장류가 시작된 이유 한 줄]
```
**예시 (된장):**
```
국이 끓기 시작하면 향이 먼저 납니다.

메주를 직접 띄우는 곳에서 왔습니다.
```
**예시 (간장):**
```
색이 맑습니다. 끓이지 않아서입니다.

3년 동안 햇빛과 바람이 농도를 만들었습니다.
```

### SD-03 · Story 카피 템플릿
```
[생산 배경 또는 생산자 이야기 — 2~3문장]

[이 상품이 어디에 어울리는가 — 1문장]
```
**예시 (된장):**
```
충북 어딘가의 마당에서 매년 겨울 메주를 띄웁니다.
공장에서 만든 메주가 아닙니다.
시간이 만든 맛입니다.

된장찌개보다 쌈장에 먼저 써보세요.
```

### SD-04 · Point 항목 작성 기준
| 항목 | 작성 기준 |
|------|-----------|
| 원재료 | 국산/수입 구분 · 콩 품종 명시 가능 시 명시 |
| 숙성 기간 | 개월 또는 년 단위로 명확히 |
| 용량 | g 또는 mL |
| 보관 방법 | 냉장/냉동/실온 + 개봉 후 권장 사용 기간 |
| 제조사/생산지 | 지역 또는 생산자명 |
| 알레르기 정보 | 해당 시 필수 |

### 상품 설명 본문 구조
```
1. 큐레이션 노트 (에디터 한 줄)
2. Hook 카피 (SD-02)
3. Story 카피 (SD-03)
4. Point 목록 (SD-04)
5. 보관·사용법 안내
6. 배송 정보
```

## 카테고리 2 · 발효 도구 (항아리·옹기)

### 상품명 작성 규칙
- **구조:** [도구 종류] [용량/크기] (예: 옹기 항아리 5L / 된장 숙성 옹기 10호)
- **금지:** "명품", "최고급", "장인의" 등 수식어
- **산지 표기:** 울산·광주 등 산지 명시 가능 시 포함
- **영문 병기 예시:** 옹기 항아리 5L · Onggi Jar 5L

### 큐레이션 노트 (SOWN 에디터 한 줄)
- **원칙:** 왜 이 항아리/옹기인가 — 호흡하는 흙의 특성·용도·만든 곳 중 하나
- **형식:** 이유가 명확해야 함. "느낌이 좋아서"는 불가
- **예시 (항아리):** "숨을 쉽니다. 옹기의 기공이 발효 중 가스를 배출하고 온도를 조절합니다. 플라스틱 용기와 결과가 다릅니다."
- **예시 (옹기 작은 것):** "처음 담그는 분을 위해 고른 크기입니다. 1~2인 가구 기준 가장 잘 맞는 용량입니다."

### SD-02 · Hook 카피 템플릿
```
[도구가 만드는 차이 — 감각·결과로 묘사]

[이 도구를 고른 이유 한 줄]
```
**예시 (항아리):**
```
항아리에 담으면 발효가 달라집니다.

흙이 숨을 쉬기 때문입니다.
```

### SD-03 · Story 카피 템플릿
```
[이 도구가 만들어지는 곳·사람 이야기 — 2~3문장]

[언제, 어떤 용도에 맞는 도구인가 — 1문장]
```
**예시 (항아리):**
```
울산 외옹치 가마에서 나왔습니다.
전기 가마가 아닌 장작 가마로 구워낸 옹기입니다.
기공이 더 고르게 열립니다.

처음 된장을 담그는 분에게 권합니다.
```

### SD-04 · Point 항목 작성 기준
| 항목 | 작성 기준 |
|------|-----------|
| 재질 | 옹기토/도자기/스테인리스 등 |
| 용량 | L 단위 |
| 크기 | 지름 × 높이 (mm 또는 cm) |
| 산지/제작처 | 지역 또는 작가명 |
| 용도 | 장류/술/김치 등 적합 용도 명시 |
| 세척 방법 | 세제 사용 가능 여부 |

### 상품 설명 본문 구조
```
1. 큐레이션 노트 (에디터 한 줄)
2. Hook 카피 (SD-02)
3. Story 카피 (SD-03)
4. Point 목록 (SD-04)
5. 사용·관리 방법
6. 배송 정보 (파손 주의 안내 포함)
```

**파손 주의 문구:**
> 옹기·도자기 상품은 충격에 약합니다. 이중 포장하여 발송하나, 파손 시 수령 후 24시간 이내 사진과 함께 문의해주세요.

## 카테고리 3 · 발효 재료 (누룩·메주콩)

### 상품명 작성 규칙
- **구조:** [재료 종류] [원산지/품종/특징] (예: 국산 메주콩 500g / 전통 입국 누룩 200g)
- **중량 포함:** 상품명에 중량 표기 권장
- **금지:** "천연", "무농약" 등 인증 없이 사용 불가
- **영문 병기 예시:** 전통 누룩 200g · Nuruk (Traditional Fermentation Starter) 200g

### 큐레이션 노트 (SOWN 에디터 한 줄)
- **원칙:** 재료가 발효에 어떤 역할을 하는지, 또는 이 산지/품종을 고른 이유
- **예시 (메주콩):** "장을 담글 때 콩의 크기가 일정해야 발효가 고르게 됩니다. 충북 단양 산지 선별콩을 고른 이유입니다."
- **예시 (누룩):** "술의 맛은 누룩이 결정합니다. 직접 띄운 전통 입국 누룩입니다. 효모가 살아있습니다."

### SD-02 · Hook 카피 템플릿
```
[이 재료의 본질 — 발효에서 하는 역할]

[이 재료가 특별한 이유 한 줄]
```
**예시 (누룩):**
```
누룩이 없으면 술이 되지 않습니다.

효모가 살아있는 누룩입니다.
```
**예시 (메주콩):**
```
장의 시작은 콩입니다.

크기가 균일할수록 발효가 고르게 됩니다.
```

### SD-03 · Story 카피 템플릿
```
[이 재료의 생산 배경 또는 생산자 이야기 — 2~3문장]

[어떤 발효에 어울리는가 — 1문장]
```
**예시 (누룩):**
```
밀을 분쇄하고 발로 밟아 성형합니다.
한 달을 띄워 효모와 곰팡이가 균형을 잡습니다.
입국이라 쌀 발효에 특히 잘 맞습니다.

탁주·청주·막걸리 모두에 쓸 수 있습니다.
```

### SD-04 · Point 항목 작성 기준
| 항목 | 작성 기준 |
|------|-----------|
| 원산지 | 국산/수입 + 지역 명시 가능 시 |
| 중량 | g 단위 |
| 주요 성분 | 콩 품종·밀 등 원재료 |
| 보관 방법 | 냉장/냉동/건조 보관 여부 |
| 사용 방법 개요 | 어떤 발효에, 얼마나 사용하는가 |
| 유통기한 | 개봉 전/후 구분 |

### 상품 설명 본문 구조
```
1. 큐레이션 노트 (에디터 한 줄)
2. Hook 카피 (SD-02)
3. Story 카피 (SD-03)
4. Point 목록 (SD-04)
5. 사용 방법 안내
6. 보관 방법
7. 배송 정보
```

## 카테고리 4 · 술 담그기 세트

> 주의: 술 직접 판매 불가. 도구·재료 세트만 판매. 상품명·카피에 "술" 판매 표현 금지.

### 상품명 작성 규칙
- **구조:** [담그기 종류] 세트 [구성품 수] (예: 전통 탁주 담그기 세트 / 막걸리 담그기 입문 세트)
- **금지:** "술 판매", "알코올 도수" 등 주류 판매 인상 표현
- **강조 포인트:** 세트 구성의 완결성 — 이 세트만 있으면 시작할 수 있다
- **영문 병기 예시:** 탁주 담그기 세트 · Takju Brewing Starter Set

### 큐레이션 노트 (SOWN 에디터 한 줄)
- **원칙:** 이 세트로 무엇을 담글 수 있는지, 왜 이 구성인지
- **예시 (탁주 세트):** "처음 담그는 분이 헤매지 않도록 구성했습니다. 쌀·누룩·항아리·체까지, 세트 하나로 첫 번째 탁주를 완성할 수 있습니다."
- **예시 (막걸리 세트):** "발효 시작점부터 걸러내는 마지막 단계까지 필요한 것을 모았습니다. 조합이 이미 검증되어 있습니다."

### SD-02 · Hook 카피 템플릿
```
[첫 담그기의 막막함을 직접 언급]

[이 세트가 그것을 해결하는 방식]
```
**예시:**
```
무엇부터 사야 할지 모르는 것이 가장 큰 장벽이었습니다.

이 세트 하나로 첫 탁주를 시작할 수 있습니다.
```

### SD-03 · Story 카피 템플릿
```
[이 세트가 만들어진 이유 — 2~3문장]

[이 세트로 무엇을 담글 수 있는가 — 1문장]
```
**예시:**
```
처음 전통주를 담그려 했을 때, 재료를 따로따로 찾아다녔습니다.
어떤 누룩이 맞는지, 항아리 크기는 어떻게 고르는지—아무도 알려주지 않았습니다.
그래서 SOWN이 먼저 골랐습니다.

이 세트로 탁주 2리터를 담글 수 있습니다.
```

### SD-04 · Point 항목 작성 기준
| 항목 | 작성 기준 |
|------|-----------|
| 구성품 목록 | 각 품목명 + 수량/중량 전부 명시 |
| 담글 수 있는 양 | 완성 후 예상 용량 (mL 또는 L) |
| 소요 시간 | 발효 기간 일수 |
| 난이도 | 입문/중급 구분 |
| 보관 방법 | 각 구성품별 보관법 |
| 포함되지 않는 것 | 쌀·물 등 직접 준비 필요 항목 명시 |

### 상품 설명 본문 구조
```
1. 큐레이션 노트 (에디터 한 줄)
2. Hook 카피 (SD-02)
3. Story 카피 (SD-03)
4. 구성품 목록 (SD-04)
5. 담그는 순서 간략 안내 (3~5단계)
6. 준비해야 할 것 (세트에 미포함 재료)
7. 보관 방법
8. 배송 정보
```

**담그는 순서 예시 (탁주):**
```
Step 1 — 쌀 씻기·불리기 (6~8시간)
Step 2 — 고두밥 짓기
Step 3 — 누룩·물 넣고 항아리에 담기
Step 4 — 발효 기다리기 (7~10일)
Step 5 — 체로 거르기
```

**직접 준비 안내 문구:**
> 이 세트에는 쌀과 물이 포함되어 있지 않습니다. 쌀 800g과 정수된 물 1.5L를 준비해주세요.

## 공통 — 배송 정보 표준 문구

```
배송: 주문 확인 후 2~3일 이내 출고 (주말·공휴일 제외)
무료배송: 50,000원 이상 / 미만 시 배송비 3,000원
식품 반품: 식품 특성상 단순 변심 반품 불가 (상품 불량·오배송 제외)
도구 반품: 수령 후 7일 이내 미개봉 원상태 반품 가능
문의: 마이페이지 → CS 문의
```
