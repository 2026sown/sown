## 2. 문제 정의

### 지금 시장에 없는 것
- 완제품 + 재료 + 도구 + 체험 원스톱 편집샵 — 국내 선례 없음
- 감성 브랜딩 발효 편집샵 — 국내 선례 없음
- 발효를 선물 경험으로 포지셔닝한 커머스 — 미개척
- 입문자 온보딩 흐름을 갖춘 발효 커머스 — 미개척

### 타겟의 페인포인트
1. **어디서 시작해야 할지 모른다** — 재료 선택지가 파편화되어 있고 첫 진입점이 없다
2. **선택지를 신뢰할 수 없다** — 품질 기준 불명확, 큐레이션 맥락 없음
3. **혼자 하기 두렵다** — 실패 시 시간·재료 손해. 확인 방법 없음

## 3. 솔루션

### SOWN이란
발효식품의 큐레이션 편집샵. "이것부터 시작하세요"라고 말하는 편집자.
재료에 맥락을 담고, 체험으로 문턱을 없앤다.

씨앗을 뿌리듯(sow) 발효를 시작하고, 그 시간이 소원(所願)이 된다.

### 경쟁사 대비 포지셔닝

| 구분 | 기존 식품 쇼핑몰 | 커뮤니티 직거래 | SOWN |
|------|----------------|----------------|------|
| 완제품 판매 | O | O | O |
| 재료·도구 큐레이션 | 일부 | X | O |
| 체험 클래스 연계 | X | X | O |
| 입문자 온보딩 | X | X | O |
| 브랜드 감성 | 없음 | 없음 | 정제·고급 |
| 선물 경험 | X | X | O |

## 4. 시장 기회

### UNESCO 등재 타이밍
2024년 12월 유네스코 장 담그기 문화 인류무형문화유산 등재.
*(출처: UNESCO 공식 발표 2024.12)*
- 장류 수출 역대 최고치 기록
- 미디어 노출 급증 → 발효 관련 관심도 상승
- 발효 브랜드가 감성으로 포지셔닝될 수 있는 최초의 타이밍

### 전통주 시장
- MZ세대가 전통주 소비 주도
- 평균 구매 단가 6만 4천원, 재구매율 40% *(출처: 세레스 리서치)*
- SOWN은 술 완제품 직판 없이 술 담그기 도구·재료로 법적 리스크 없이 진입

### 글로벌 발효식품 시장
- 2024년 기준 1,265억 달러, CAGR 7% 성장 *(출처: GM Insights)*

### 법적 구조
- 장류 온라인 직판: 제약 없음
- 발효 도구·재료 온라인 직판: 제약 없음
- 술 완제품 직판: 불가 → 양조 도구·재료로 대체

## 5. 타겟

## 9. 기대 효과

### UNESCO 등재 타이밍
등재 확정 시점 전후 미디어 집중 노출. 이 시기 브랜드·유통 채널 선구축 시 트래픽 흡수 우위 선점. 경쟁사 동일 포지셔닝 시도 시 최소 6~12개월 격차 발생.
*(등재 후 시장 규모 확대 수치 — 데이터 없음)*

### 공백 시장 선점
- 입문자 키트: 쿠팡·네이버·컬리 검색 시 풀세트 키트 공백 확인
- 선물 시장: 홈쇼핑 위주로만 형성, D2C 브랜딩 선물세트 부재
- 계절 집중(1~3월) 입문자 유입 온보딩 구조로 흡수 가능
*(입문자 비율, 선물 시장 규모 수치 — 데이터 없음)*

### 커머스 + 클래스 시너지
클래스 수강 → 커머스 전환 / 커머스 구매 → 클래스 유입 순환 구조.
*(전환율·LTV 수치 — 데이터 없음, 론칭 후 측정)*
